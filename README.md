# App Lock

An uncrackable app lock with biometric authentication, PIN protection, and intruder detection.

## Features

- **App Protection**: Select and protect any app with PIN or biometric authentication
- **Biometric Unlock**: Use fingerprint or Face ID for quick access
- **Intruder Detection**: Capture photos of unauthorized access attempts
- **Progressive Lockout**: Increasing delays after failed attempts
- **Security Settings**: Screenshot prevention, root detection, and more

## Tech Stack

- **Frontend**: React Native with Expo
- **Backend**: Express.js
- **Storage**: AsyncStorage for local data persistence
- **Authentication**: expo-local-authentication for biometrics

## Getting Started

1. Clone the repository
2. Run `npm install`
3. Run `npm run expo:dev` for the frontend
4. Run `npm run server:dev` for the backend
